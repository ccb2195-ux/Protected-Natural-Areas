# Protected-Natural-Areas
A Data Story on which countries are best converving their land and sea terriroty. 

## Process:

So I thought about doing this after stumbling on the 2023 World Bank Sustainability Goals document and saw the awesome graphics they had on there. I wanted to explore the dataset and see if I could pull something intersting out of it as well. I figured it might be nice to dip my toes into climate/conservation data as well.

1. I got my first dataset from Protected Plant, and I was able to pull shape files, sizes, datas, and more for every known protected environment on the planet.
2. I calculated the largest and small, as well as any major outliers in countries, but figured that the countries with the most protected land also happened to be the biggest. I found it more informative to see who used their land the best by percentage preotected.
3. I graphed my charts in matplotlib and then exported them to Illistrator to clean and build responsive layers.
4. I had claude help me with the scrollytelling and html build out by providing it with a text file containing detailed instructions on how I wanted to site to look.
